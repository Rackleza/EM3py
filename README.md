# My Markdown Guide

Markdown is a super simple way to format plain text. It lets you add things like headers, bold text, lists, and links without writing complex HTML.

---

## 1. Headers
To make a heading, just put a hashtag (`#`) in front of your text. The more hashtags you add, the smaller the heading gets (up to six).

# This is a main title (H1)
## This is a major section (H2)
### This is a sub-section (H3)

---

## 2. Emphasis
You can make text bold or italic using asterisks (`*`) or underscores (`_`).

This text is *italicized*.
This text is **bolded**.
This text is ***both bold and italic***.

---

## 3. Lists
For bullet points, use a dash or an asterisk. For numbered lists, just use numbers.

* Blueberry
* Strawberry
* Raspberry

1. Wake up
2. Make coffee
3. Code Markdown

---

## 4. Links and Images
The syntax for links and images is very similar. Think of it as `[text](url)`.

Check out [Google](https://www.google.com).

To add an image, just put an exclamation mark `!` right at the front:
![Placeholder Image](https://via.placeholder.com/150)

---

## 5. Code Blocks
If you want to show code inside your markdown file without it actually executing, wrap it in backticks.

To print text in Python, use the `print()` function.

For longer blocks of code, use three backticks before and after:

```python
def hello_world():
    print("Hello, World!")
```
